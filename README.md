# 🎙️ Hey Khushboo - Smart Voice Assistant with Session Memory

A lightweight, local-first voice assistant built using Python, Gradio, and Google Speech Recognition. Developed to solve optimization challenges by handling routine queries locally and maintaining a dynamic session memory array.

## ✨ Key Features
- **Voice UI:** Responsive web interface built with Gradio featuring automatic audio playback.
- **Session Memory:** Implements a conversation history array to remember user context.
- **Local Fallbacks:** Instantly answers time, weather, and specific facts locally in Python to save API tokens.

## 🛠️ Tech Stack
Python | Gradio | SpeechRecognition | gTTS | Pytz
